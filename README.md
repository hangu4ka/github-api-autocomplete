GitHub API Autocomplete
A web application that searches GitHub repositories in real time using the GitHub Search API.

🔗 Demo
https://hangu4ka.github.io/github-api-autocomplete/


How to Use
Start typing a repository name in the search field

Select a repository from the autocomplete dropdown

The repository is added to the list below with its details

Click ✕ to remove a repository from the list

Technologies
HTML / CSS / JavaScript (ES6+)

GitHub Search API

Fetch API + async/await

Debounce pattern
